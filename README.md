Project: Agentic AWS Security Lab

Goal:
Build AWS security scanners and an AI security analyst that can inspect IAM, S3, CloudTrail, GuardDuty, and network risks, then generate evidence-backed incident reports.

Final features:
- IAM risk scanner
- S3 security scanner
- CloudTrail event parser
- GuardDuty finding analyzer
- Security group scanner
- Agentic analyst with tool-calling
- Playbook-based remediation guidance
- Markdown incident reports

Safety rule:
All tools run read-only by default. No destructive AWS changes without human approval.
